# Balance Checklist

## Progression

- [ ] Band identity is explicit and visible to players.
- [ ] Neighboring levels are not near-clones.
- [ ] Special threats are introduced before being combined.

## Economy

- [ ] Start resources allow stable opener.
- [ ] Economy plants can be established before heavy waves.
- [ ] Recovery is possible after one lane mistake.

## Pressure

- [ ] Spike windows are short and intentional.
- [ ] Quiet windows exist for rebuild.
- [ ] Combined pressure is bounded in early/mid bands.

## Endgame

- [ ] Final levels are harder by composition and pacing, not only raw inflation.
- [ ] Boss level has unique identity and telegraphed threat.
